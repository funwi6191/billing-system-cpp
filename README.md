# 💳 Simple Billing System (C++)

A beginner-friendly **Billing System** built with C++. The program allows users to add products, calculate totals, apply discounts, and generate a simple bill. Designed to demonstrate object-oriented programming and vectors.

---

## 🚀 Features
- Add products with name, quantity, and price
- Calculate total bill
- Apply discounts or taxes
- Display final invoice
